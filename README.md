Software Testing Automation Project

This project demonstrates automated web testing using Selenium WebDriver and Java. The objective of the project is to automate browser actions, execute test cases, and validate web application functionality efficiently.

Features
Automated browser testing
Web element identification and interaction
Test case execution using Selenium WebDriver
Cross-browser automation support
Maven dependency management
Improved testing accuracy and efficiency
Technologies Used
Java
Selenium WebDriver
Maven
ChromeDriver
Visual Studio Code
